The [Brave Search MCP Server](https://github.com/brave/brave-search-mcp-server) gives your AI agent live access to the web. Whenever your agent needs current information â€” recent library releases, API documentation, error messages, or local business details â€” it searches Brave and returns real results instead of guessing.

It exposes six dedicated tools:

- **Web search** â€” full-featured search with filtering and spellcheck
- **Local search** â€” business and place discovery with ratings
- **Video search** â€” video results with metadata and thumbnails
- **Image search** â€” image discovery with automatic fetching
- **News search** â€” current articles with freshness controls
- **AI summarizer** â€” condense search results into a concise answer

This eliminates a whole class of hallucinations around version numbers, API changes, and recent events.

### Setup

1. Grab an API key from **brave.com/search/api** â€” the free tier gives you 2,000 queries/month
2. Add the server to your MCP config and set the `BRAVE_API_KEY` environment variable
3. Install with `npx -y @brave/brave-search-mcp-server`, or use Docker via `mcp/brave-search`

The default transport is STDIO. Switch to HTTP with `BRAVE_MCP_TRANSPORT=http` if needed. You can also control which tools are exposed using `BRAVE_MCP_ENABLED_TOOLS` or `BRAVE_MCP_DISABLED_TOOLS` to limit the surface area.
