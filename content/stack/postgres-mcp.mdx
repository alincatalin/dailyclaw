### Setup

1. Prepare a connection string: `postgresql://user:password@host:port/database`
2. Add the server to your MCP config with the `POSTGRES_CONNECTION_STRING` environment variable
3. Run `npx @modelcontextprotocol/server-postgres` to verify

For production databases, use a **read-only connection string** to enforce safe access at the database level.
