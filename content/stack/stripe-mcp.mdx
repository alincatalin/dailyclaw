### Setup

1. Grab your API key from **Developers > API keys** in the Stripe dashboard
2. Use a **restricted key** with only the permissions you need â€” not your secret key
3. Add the server to your MCP config with the `STRIPE_API_KEY` environment variable
4. Run `npx @stripe/mcp-server` to test the connection

Start with **test mode keys** first to verify everything works before pointing it at live data.
