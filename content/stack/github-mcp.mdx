The GitHub MCP Server connects your AI agent to the full GitHub API surface. This is one of the most broadly adopted MCP servers because nearly every developer workflow touches GitHub at some point.

Setup takes about two minutes. Generate a GitHub Personal Access Token (classic or fine-grained) from **Settings > Developer settings > Personal access tokens**. Grant it the scopes you need â€” `repo` for private repositories, `read:org` for organization data. Add the server to your MCP config with the `GITHUB_PERSONAL_ACCESS_TOKEN` environment variable. Run `npx @modelcontextprotocol/server-github` to confirm it connects. Fine-grained tokens are recommended if you want to limit access to specific repositories.
